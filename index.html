This is a simple lightbox.

Just using a little bit of Javascript and Css.

This is the markup : 

		
<div id="Lightbox">
	<img src="">
</div>

You have to put it somewhere in your html.

Make sure you have no other fixed element over z-index 1000 or change this value in lightbox.css

And this is how you have to wrap your images :

		
<div id="Lightbox">
	<a class="lightbox" href="dog.jpg">
		<img src="dog.jpg" alt="Your alt attribute">
	</a>
</div>
